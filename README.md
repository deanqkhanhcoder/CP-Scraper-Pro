# 🚀 CP Scraper Pro là gì?

CP Scraper Pro là một bookmarklet dành cho lập trình thi đấu (Competitive Programming), giúp tự động hóa toàn bộ workflow khi học và phân tích bài toán CP bằng AI.

Thay vì:

* copy link thủ công
* format prompt
* tự viết context cho AI
* dọn URL rác
* copy statement nhiều lần

Bạn chỉ cần:

```text
Mở đề bài → Bấm bookmarklet → Paste vào AI
```

CP Scraper Pro sẽ tự động:

* nhận diện nền tảng
* chuẩn hóa URL
* thu thập metadata
* tạo structured prompt
* hỗ trợ workflow với AI

## 🧠 Chuẩn hóa URL thông minh

Tự động:

* loại bỏ tracking params
* xóa referral
* canonicalize URL
* chuẩn hóa link bài toán

Ví dụ:

```text

→ canonical URL sạch
```
